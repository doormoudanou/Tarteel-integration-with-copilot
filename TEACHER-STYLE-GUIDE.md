## 🎤 Recording Tips

### For Best Results:
1. **Listen completely** - Don't skip the Ustaz recitation
2. **Wait for cue** - Press button when ready
3. **Speak clearly** - Enunciate each letter
4. **Natural pace** - Don't rush or go too slow
5. **Proper Tajweed** - Apply rules you learned
6. **One attempt** - Record the whole ayah at once

### Common Issues & Fixes:

| Problem | Solution |
|---------|----------|
| Low confidence | Speak more clearly, slower pace |
| Makhraj errors | Focus on letter articulation point |
| Madd errors | Hold vowels longer |
| Low volume | Check microphone level |
| Noise issues | Use quiet room |
